Twitter/X 上的终极屏蔽工具。为时间线中的每条帖子添加一键屏蔽与隐藏按钮。界面风格与 X 原生 UI 自然融合，使用起来没有违和感。

■ 主要功能

- 一键屏蔽 — 点击一次即可立即屏蔽不想看到的用户
- 一键隐藏 — 保持关注关系不变，同时不再在时间线上显示该用户的内容
- 转帖支持 — 可分别对转帖者和原作者执行屏蔽或隐藏
- 引用帖子支持 — 可直接对引用帖子中的用户执行操作
- 状态保持 — 页面切换后，已屏蔽的帖子仍会自动隐藏

■ 按钮显示位置

- 时间线帖子
- 转帖（转帖者 + 原作者）
- 引用帖子
- 个人资料页面
- 推荐用户（You might like）
- 关注 / 关注者列表
- 悬浮卡片（用户弹出卡片）

■ 特点

- 与 X 原生 UI 无缝融合的设计
- 轻量且快速，不影响页面性能
- 可分别控制屏蔽按钮和隐藏按钮的显示 / 隐藏
- 可选：在屏蔽已关注用户前先确认
- 统计功能 — 记录屏蔽与隐藏次数
- 支持日本語 / English / 简体中文
